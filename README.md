# BookNest

A personal library management web application built with HTML, CSS, and JavaScript.

## Features

- Add books dynamically
- Delete books
- Form validation
- Real-time book counter
- Modern responsive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)
